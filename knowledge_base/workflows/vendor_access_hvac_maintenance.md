# Workflow: HVAC Vendor Access & Verification
**Document ID:** VEND-HVAC-02
**Category:** GateX Access Control

## 1. Pre-Arrival Requirements
All HVAC vendors must upload a valid **Certificate of Insurance (COI)** to the vendor portal 24 hours before the scheduled visit.
* **Minimum Liability:** $1,000,000 general liability.
* **Endorsement:** Must list "PropFlow Management LLC" as an additional insured.

## 2. On-Site Check-In (GateX Agent Protocol)
When the vendor arrives at the gate/lobby:
1.  **Identity Verification:** Vendor must present a photo ID matching the work order.
2.  **Key Handoff:** GateX authorizes the release of the "Maintenance Master Key" (Key Set B).
3.  **Time Logging:** Vendor check-in time is logged automatically.

## 3. Service Completion
Upon finishing the job, the vendor must:
1.  **Thermostat Check:** Set thermostat to 72°F (Cool) or 68°F (Heat) before leaving.
2.  **Filter Photo:** Upload a timestamped photo of the new air filter installed.
3.  **Check-Out:** Return Key Set B. If the key is not returned within 4 hours, an alert is sent to the Property Manager.
