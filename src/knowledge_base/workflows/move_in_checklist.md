# Workflow: Tenant Move-In Inspection
**Document ID:** WORK-MOVEIN-01

## 1. Key Handoff
* Verify First Month Rent & Security Deposit have cleared.
* Issue: 2 Unit Keys, 1 Mailbox Key, 1 Garage Fob.
* Have tenant sign "Key Receipt Form."

## 2. Unit Condition Walkthrough
GateX Agent / Property Manager must verify:
* [ ] Smoke detectors tested (Green light).
* [ ] HVAC filter is new/clean.
* [ ] All faucets run hot/cold with no leaks.
* [ ] Toilets flush completely.
* [ ] Appliances turn on (Stove, Fridge, Dishwasher).
* [ ] Photos taken of any pre-existing scratches on floors/walls.

## 3. Utility Transfer
* Verify tenant has switched electricity to their name (Confirm # required).
* Provide Wi-Fi setup instructions.
