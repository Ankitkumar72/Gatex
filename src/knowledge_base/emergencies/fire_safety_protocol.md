# Emergency Protocol: Fire Safety & Evacuation
**Document ID:** EMER-FIRE-001
**Severity:** Critical (Life Threatening)

## 1. Immediate Action
In the event of visible fire or smoke:
1.  **Activate Alarm:** Pull the nearest manual fire alarm lever.
2.  **Evacuate:** Leave the building immediately using the stairs. **DO NOT USE ELEVATORS.**
3.  **Call 911:** Once safe, call emergency services.

## 2. False Alarms
If a smoke detector chirps intermittently (no fire):
* This usually indicates a low battery.
* **Tenant Action:** Replace the 9V battery immediately.
* **Maintenance:** If replacing the battery does not stop the chirping, submit a "Routine" maintenance request.

## 3. Fire Extinguishers
* Locations: Under kitchen sink (Unit), Hallway every 50ft (Common Area).
* Usage: Only attempt to extinguish small, contained fires (trash can size). For anything larger, evacuate.
